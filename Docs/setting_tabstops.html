# Setting COBOL tab stops

IBM Z Open Editor provides support for adding custom tab stops in your COBOL source code. You can provide a column number (0-based columns) to stop the cursor when you press the tab key. By default, IBM Z Open Editor sets the following tab stops for COBOL: 0, 6, 7, 11.

To configure custom COBOL tab stops:

"zopeneditor.cobol.tabstops": [
    0,
    6,
    7,
    11,
    15,
    19,
    72
]
